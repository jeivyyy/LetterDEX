About the project

LetterDEX

LetterDEX is a visual Pokédex built with Angular, focused on Pokémon forms, type identity, base stats, comments and special transformations like Mega Evolution and Gigantamax.

What is LetterDEX?

LetterDEX is a web project created to organize Pokémon information in a more visual and interactive way. The application separates the classic Pokédex, Mega Evolutions and Gigantamax forms into their own pages, keeping the same card-based style across the project.

151

Pokédex

The Pokédex page displays the first 151 Pokémon, using data from the PokeAPI to show names, sprites, numbers and types.

M

Mega Evolution

The Mega page lists Mega Evolution forms with a dedicated card style and the Mega Evolution symbol placed at the bottom of each card.

G

Gigantamax

The G-Max page follows the same visual structure as the Mega page, showing Gigantamax forms with a dedicated Dynamax/Gigantamax icon.

#

Comments

The details page includes a comment section where users can write opinions and organize them using tags like #META, #LOVE, #HATE and #FUNFACT.

API

PokeAPI

Pokémon data is fetched from the PokeAPI, including sprites, types, abilities, forms and base stats used in the details page.

UX

Visual Identity

The interface uses rounded cards, soft shadows, 3D buttons, type backgrounds, status sliders and animated assets to create a playful but organized visual experience.

Built with
